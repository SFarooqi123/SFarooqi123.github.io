Project 0: Becoming Friends with Your Camera

A hands-on warm-up exploring focal length, distance, and motion for perspective control.

Overview

This project builds an intuition for how distance to subject, zoom, and camera motion alter the perceived shape and spatial relationships in a scene. The activities below are structured so you can drop your images in later—no code changes required—just replace the media/*.jpg files.

Tip: keep framing consistent when comparing shots (same subject size or same camera position) so differences are attributable to the parameter you’re testing.

Part 1Self-Portraits: Close vs. Stepped-Back

Compare a tight close-up with a stepped-back shot (keeping the head the same size via zoom). This highlights how distance and focal length influence facial proportions.

Part 2Architectural Perspective & Compression

Shoot the same street/scene twice: (A) from far away with zoom, and (B) from up close with a wider lens. Observe vanishing lines and relative object spacing.

Part 3The Dolly Zoom (Vertigo Effect)

Create a short sequence where you physically move backward while zooming in (or vice-versa) to keep the subject roughly the same size while the background “breathes.” A single representative frame is shown below; you can also embed a GIF or video later.

Methods Setup Notes

Each experiment isolates one variable at a time:

  • Part 1: Keep head size consistent between shots; vary camera distance + focal length together.
  • Part 2: Match scene center; shoot from near (wide) vs. far (tele) to compare perspective lines.
  • Part 3: Maintain subject size by counterbalancing motion with zoom; capture 10–20 frames or a short clip.

File layout assumes images live in ./media/. Replace img1.jpg … img5.jpg with your own filenames or extend the figure blocks to add more comparisons.
